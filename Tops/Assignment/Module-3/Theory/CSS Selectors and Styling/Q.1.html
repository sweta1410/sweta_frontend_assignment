What is a CSS selector? Provide examples of element, class, and ID selectors. Ans. A CSS selector is a pattern used to select and style HTML elements based on their attributes, relationships, or position in the document. Examples of CSS Selectors: *Element Selector:- Targets all elements of a specific type. css p { color: blue; } Styles all

(paragraph) elements with blue text. *Class Selector: Targets elements with a specific class name. css .highlight { background-color: yellow; } Styles all elements that have the class highlight with a yellow background. Example in HTML: html

This is highlighted text.

*ID Selector: Targets a specific element using its unique id attribute. css #main-title { font-size: 24px; font-weight: bold; } Styles the element with the ID main-title by increasing its font size and making it bold. Example in HTML: html

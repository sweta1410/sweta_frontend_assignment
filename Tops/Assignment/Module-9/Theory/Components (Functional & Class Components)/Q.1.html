What are components in React? Explain the difference between functional components and class components Ans. In React, components are the fundamental building blocks for creating user interfaces. They are reusable, self-contained pieces of UI that can be combined to build complex applications. Components receive data via "props" and return React elements, which describe what should be displayed on the screen. * Functional component:- These are simpler components defined as plain JavaScript functions. They accept props as an argument and return JSX (JavaScript XML) to describe the UI. With the introduction of React Hooks in version 16.8, functional components can now manage state and handle side effects, capabilities previously exclusive to class components.

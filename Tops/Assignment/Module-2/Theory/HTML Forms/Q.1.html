What are HTML forms used for? Describe the purpose of the input, textarea, select, and button elements. Ans. HTML Forms and Their Purpose HTML forms are used to collect user input and send it to a server for processing. They provide a structured way to gather data such as text, numbers, selections, and actions from users. Forms are commonly used in login pages, search bars, contact forms, and online orders. * Elements and Their Purpose;- :- The element is used to collect various types of data such as text, numbers, emails, passwords, and more. Common type attributes: text - Single-line text input password - Masks the entered text email - Validates email format number - Allows only numerical values checkbox - Allows multiple selections radio - Allows a single selection within a group submit - Sends form data to the server

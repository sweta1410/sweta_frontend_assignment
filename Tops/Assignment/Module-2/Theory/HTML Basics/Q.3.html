What is the difference between block-level elements and inline elements in HTML? Provide examples of each. Ans. Block-Level Elements - These elements take up the full width available (by default) and start on a new line. - They create a "block" or a distinct section in the document. - Commonly used for structuring the layout of a webpage. * Examples of Block-Level Elements:

Create Github account & make repository. Ans. * Create a GitHub Account:- 1. Go to GitHub Website: Open your browser and visit https://github.com. 2. Sign Up: Click on the Sign up button. 3. Enter Your Details. -Username -Email address -Password 4. Verify Your Email. Open the email and click the verification link. 5. Complete the Setup. * Create a Repository Once you have signed in to GitHub then follow these steps:- - Go to Your Dashboard: After logging in you'll be taken to your GitHub dashboard. - Create a New Repository: - Click the "+" icon in the upper-right corner of the screen (next to your profile picture). Select "New repository" from the dropdown. - Fill in Repository Details: Repository Name: Choose a name for your repository (e.g., my-first-repo). Description (optional): Add a short description of the repository. Public or Private: Choose whether you want your repository to be public or private. Initialize Repository with a README: You can check this option if you want GitHub to create a README.md file for you. Create the Repository: Click the Create repository button. Now, you have successfully created your GitHub account and repository. If you want to add files to your repository or clone it to your system.
